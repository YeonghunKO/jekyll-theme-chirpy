# scss 코딩

우선 extend 와 variables 를 설정하자.

```scss
%miniTitle {
  font-family: "Montserrat";
  font-weight: 500;
  font-size: 15px;
  text-transform: uppercase;
  letter-spacing: 2px;
}
```

```scss
$bg: #f3ede8;
$sideSpace: 140px;
```

이렇게 일단 설정해놓고 styles.scss 는 다음 글에서 알아보자 글이 너무 길어질 것 같으니.
